Schema Document Properties

Target Namespace http://qifstandards.org/xsd/qif3
Version 3.0.0
Element and Attribute Namespaces
  • Global element and attribute declarations belong to this schema's target namespace.
  • By default, local element declarations belong to this schema's target namespace.
  • By default, local attribute declarations have no namespace.
Schema Composition
  • This schema includes components from the following schema document(s):
    • ../QIFLibrary/IntermediatesPMI.xsd
    • ../QIFLibrary/GenericExpressions.xsd
Documentation The QIFPlan.xsd schema file defines the global Plan element, which is the root of a measurement plan. The schema file also contains the definitions of some of the types used in a Plan and a number of key/keyref pairs that constrain references used in a Plan. In all keys, the items that are collected must be unique within the collection.
