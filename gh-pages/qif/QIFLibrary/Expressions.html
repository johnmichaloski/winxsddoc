Schema Document Properties

Target Namespace http://qifstandards.org/xsd/qif3
Version 3.0.0
Element and Attribute Namespaces
  • Global element and attribute declarations belong to this schema's target namespace.
  • By default, local element declarations belong to this schema's target namespace.
  • By default, local attribute declarations have no namespace.
Schema Composition
  • This schema includes components from the following schema document(s):
    • ../QIFLibrary/GenericExpressions.xsd
    • ../QIFLibrary/PrimitivesPMI.xsd
Documentation This file defines QIF-specific Boolean expressions and arithmetic expressions. The initial intent is that these expressions should support writing QIF rules. The file includes specialized Boolean expressions for identifying the types of features and characteristics. It is assumed that a system consuming an instance file containing expressions will be capable of evaluating the expressions.
