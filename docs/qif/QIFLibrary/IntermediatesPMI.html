Name AlignmentOperationBaseType
Abstract yes
Documentation The AlignmentOperationBaseType is the abstract base type that defines a coordinate system alignment operation with a sequence number for ordering. Each alignment operation takes place in the current coordinate system, which is established as follows. 1. If a previous alignment operation exists, the current coordinate system is the coordinate system established by the previous alignment operation. 2. Otherwise, if the base coordinate system is defined, the current coordinate system is the base coordinate system. 3. Otherwise, the current coordinate system is the common coordinate system.
